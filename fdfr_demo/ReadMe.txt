Face detection and face recognition

Notes:
This is a human face detection demo program.
Please run dl_3rd.sh to get 3rd modules.
Input "make -j4" command will build this program.
Put "fdfr_demo","fdfr_demo_env.conf" to bm1880 edb board. 
The "bm1880_test_tools/video_display" folder, containing the display.py, run on your ubuntu pc as server.

More please refer to: https://developer.sophon.cn/thread/110.html
